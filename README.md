# JavaScript

A code repository for learning JavaScript. This repo contains small, focused exercises and examples organized by topic to help practice core JavaScript concepts.

## Status (what's done so far)
- Basics: 01_basics — exercises and examples covering variables, data types, simple functions.
- More Basics: 02_basics — additional beginner topics and examples.
- Advanced Basics: 03_basics — further practice problems and examples.
- Control Flow: 04_controlFlow — if/else, switch, and branching examples.
- Iterations: 05_iterations — loops, iteration patterns, and related exercises.

Other repository files/directories:
- .devcontainer — development container configuration
- .github — GitHub configuration files

## Project structure
Each numbered folder contains short, self-contained examples or exercises. Filenames are descriptive to make it easy to find topics.

## How to use
1. Clone the repository:
   git clone https://github.com/AkaPranav/JavaScript.git
2. Open the folder for the topic you want to practice (for example, `01_basics`).
3. Run the example files using Node.js (if applicable):
   node example.js
